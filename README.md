# Schwab
Charles Schwab's Coding Exercise

路 Build a responsive module on a webpage containing 4 square boxes with colors: red, green, blue, white.

路 Each box has a maximum width of 230px, and the webpage has a maximum width of 980px.

路 Each box contains a button, and when clicked changes the color of the other boxes to its color, except for the white box.

路 The white box always remains white and when its button is clicked, it resets the colors of the other boxes to their original colors.
